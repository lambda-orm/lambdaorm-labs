# keycloak

## Db

```sql
CREATE ROLE keycloak SUPERUSER NOCREATEDB NOCREATEROLE INHERIT LOGIN PASSWORD 'keycloak';
```

## Docker

Up:

``` sh
docker compose -f ./docker compose.yaml up -d 
```

``` sh
docker compose -f ./docker compose.yaml down --remove-orphans
```
